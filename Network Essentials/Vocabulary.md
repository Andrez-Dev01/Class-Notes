
# Chapter 1 Introduction
###  Networks
a group of computers and other devices that connect by some type of transmission media.
### Topology
how parts of a whole network works together
### Physical Topology 
a networks hardware and how devices work together to form the physical network
### Logical Topology
how software controls access to network resources and specific resources like applications and databases are shared on the network
### NOS (network operating system)
controls access to the entire network
### Server Operating Systems
an operating system designed to run on a server and provide network services to networked clients
### Networking Software 
a network managed through the command line (CLI)
### Peer-to-Peer model
A network in which every computer can communicate directly with every other computer.
